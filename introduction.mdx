## What is Visual Studio Code?

Visual Studio Code is a lightweight but powerful source code editor that runs on your desktop and is available for Windows, macOS, and Linux. It comes with built-in support for JavaScript, TypeScript, and Node.js and has a rich ecosystem of extensions for other languages and runtimes (such as C++, C#, Java, Python, PHP, Go, .NET).

## Why Choose VS Code?

<AccordionGroup>
  <Accordion title="Lightweight and Fast">
    VS Code starts up quickly and has a small memory footprint, making it ideal for everyday use even on less powerful machines.
  </Accordion>
  <Accordion title="Cross-Platform">
    Available for Windows, macOS, and Linux with the same experience across all platforms.
  </Accordion>
  <Accordion title="Extensible">
    Customize every aspect of VS Code through extensions, themes, and settings to match your workflow.
  </Accordion>
  <Accordion title="Open Source">
